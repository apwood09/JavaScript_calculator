# JavaScript calculator

JavaScript programming a calculator simulator to perform basic operations and keep track of all calculations

## Features 

* **Basic Mathmatics:** addition (+), subtraction (-), multiplication (*) and division (/). 
* **Responsive Design:** Works in an IDE (e.g VS code ...) terminal. 
* **Formula Logic:** Calculator uses basic mathical equations and returns a result. Stores history of mathmatical equations and results.
